# Project Management System

A simple project progress management system with a Node.js/Express backend and static frontend views for projects, progress reports, and dashboard-style summaries.

## Features

- Project creation and status tracking
- Progress report submission
- Dashboard and report views
- Express API backend

## Development

```bash
npm install
npm start
```

For development with auto-restart:

```bash
npm run dev
```

## Repository hygiene

`node_modules/` is intentionally ignored and should be installed locally with `npm install`.
